# Resources

Curated external sites and tools for finding software, tracking versions, and day-to-day development. Companion to [`applications.md`](applications.md) (curated app picks) and [`casks.md`](casks.md).

## Finding software and alternatives

| Site | Use |
| --- | --- |
| [AlternativeTo](https://alternativeto.net) | Alternatives to any app, site, or game; filter by License (Open Source / Free) and platform. |
| [OpenSourceAlternative.to](https://www.opensourcealternative.to) | Strictly open-source replacements for proprietary SaaS. |
| [opensource.builders](https://opensource.builders) | Browsable catalog of open-source alternatives to proprietary apps. |
| [awesome](https://github.com/sindresorhus/awesome) | Hub of curated "awesome" lists per topic. |
| [awesome-selfhosted](https://github.com/awesome-selfhosted/awesome-selfhosted) | Self-hostable free and open-source software. |

## Versions, releases and end-of-life

| Site | Use |
| --- | --- |
| [endoflife.date](https://endoflife.date) | Support and EOL timelines for OSes, runtimes, and tools. |
| [repology.org](https://repology.org) | Package versions across every distro and package manager. |
| [release-monitoring.org](https://release-monitoring.org) | Upstream new-release tracking (Anitya). |

## Packages and extensions

| Site | Use |
| --- | --- |
| [Homebrew Formulae](https://formulae.brew.sh) | Search brew formulae and casks. |
| [pkgs.org](https://pkgs.org) | Search Linux packages across distros (Debian, Ubuntu, Fedora, Arch, openSUSE). |
| [Flathub](https://flathub.org) | The Flatpak app store; this repo's Linux GUI fallback source. |
| [Open VSX](https://open-vsx.org) | The VSCodium extension registry this repo targets. |

## Command-line reference

| Site | Use |
| --- | --- |
| [tldr pages](https://tldr.sh) | Example-driven man-page summaries (this repo ships the `tlrc` client). |
| [cheat.sh](https://cheat.sh) | `curl cheat.sh/<cmd>` one-line cheatsheets. |
| [explainshell](https://explainshell.com) | Breaks down any shell command into its parts. |
| [ShellCheck](https://www.shellcheck.net) | Paste-and-lint shell scripts (this repo runs `shellcheck`). |
| [crontab.guru](https://crontab.guru) | Cron expression builder and explainer. |
| [regex101](https://regex101.com) | Regex tester and debugger. |
